# 🖥️ Computer Architecture 3D

An interactive, WebGL-based educational tool designed to help students visualize and understand the internal components of a computer. 

Built with **Three.js**, this application provides a 3D environment where users can explore the CPU, RAM, GPU, and SSD, observe real-time data flow animations, and learn through detailed technical specifications.

## 🚀 Key Features

- **Interactive 3D Scene**: Rotate, zoom, and navigate through a futuristic computer architecture layout.
- **Data Flow Particles**: Visual representation of data moving between components via simulated data buses.
- **Component Inspection**: Click on any part (CPU, RAM, GPU, SSD) to see detailed technical descriptions and specifications.
- **Real-Time Dashboard**: Monitor simulated system stats like CPU temperature, RAM utilization, and bus speeds.
- **System Activity Log**: A live log showing simulated background operations of the computer.
- **Futuristic UI**: Glassmorphism design with a high-tech "Cyberpunk" aesthetic.
- **Audio Feedback**: Subtle futuristic sound effects for UI interactions.

## 🛠️ Technology Stack

- **Core**: Three.js (WebGL)
- **Environment**: Vite
- **Logic**: Vanilla JavaScript
- **Styling**: CSS3 (Glassmorphism, Flexbox, Animations)
- **Fonts**: Orbitron & Inter (Google Fonts)

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sambee-coding/computer-architecture-3d.git
   ```

2. **Install dependencies**:
   ```bash
   cd computer-architecture-3d
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
